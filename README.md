# Smart Gas Leakage and Temperature Monitoring System

## Overview

This project is an Arduino-based safety monitoring system developed using Tinkercad. The system continuously monitors temperature and gas concentration levels and provides alerts when unsafe conditions are detected.

## Features

* Real-time temperature monitoring
* Gas leakage detection
* LED alert for high temperature conditions
* Buzzer alert for gas leakage detection
* Serial Monitor output for sensor readings
* Low-cost embedded safety solution

## Components Used

* Arduino Uno
* LM35 Temperature Sensor
* MQ-2 Gas Sensor
* LED
* Piezo Buzzer
* Resistors
* Breadboard

## Working Principle

1. The LM35 sensor measures ambient temperature.
2. The MQ-2 sensor detects gas concentration levels.
3. If temperature exceeds 80°C, the LED turns ON.
4. If gas concentration exceeds the threshold value, the buzzer activates.
5. Sensor readings are continuously displayed through the Serial Monitor.
